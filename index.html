3

Trajet

Calculez votre temps de trajet domicile → travail avec OpenStreetMap. Le résultat remplit automatiquement le champ « Trajet » du réveil.

Nécessaire uniquement pour les transports en commun. Gratuite, sans carte bancaire : créez un compte sur navitia.io, récupérez votre clé par email, puis collez-la ici. Elle reste enregistrée uniquement sur votre appareil.

Itinéraire et carte fournis gratuitement par OpenStreetMap — aucune clé API, aucun coût.

4

Réveil

Ajoute automatiquement quelques minutes de marge les jours où pluie, neige ou verglas sont prévus à l'heure de votre trajet (prévisions Open-Meteo, gratuites, sans clé). Nécessite l'adresse de départ renseignée dans la carte Trajet.

5

Calendrier

Le fichier .ics ajoute vos prises de poste et une alerte à l'heure de réveil calculée.

Sur iPhone, les notifications d'une app web ne sont pas fiables. Préférez toujours l'export .ics dans Calendrier.

Configurer le push (avancé, fonctionne même app fermée)

Utilise le serveur d'analyse renseigné à l'étape 1 (même URL et même code secret) pour recevoir de vraies notifications même quand l'app est fermée. Laisse l'URL vide là-bas pour garder les rappels locaux uniquement (uniquement quand l'app est ouverte).
